무언가를 입력, 선택, 자료를 전송하게 만드는 기능을 가진 요소

이런 것을 또 다른 말로는 양방향 전송 방식이라고 한다.
양방향 전송방식: 입력 - 보내기 - 받아서 처리 - 결과를 피드백

e.g. 검색, 로그인, 회원가입, 구매, 게시판, etc. 에서 폼태그가 사용됨.

  1. 검색하기 위해 필요한 단어/문장을 작성 (-> 글을 작성할 수 있는 영역이 필요)
  2. 전송버튼을 클릭 (-> a는 data를 보내는 능력 x, data를 보내는 능력을 가진 버튼)
  3. 데이터를 전송받을 곳으로 전송 (-> 어디로 보낼지 체크, 공개/비공개/삭제/수정/... 등의 대한 옵션을 걸어야한다.)
  4. 전송받은 곳에서 무언가 수행(backend)
  5. 수행에 대한 결과를 피드백 (다시 앞단에 보내줌)
  6. 피드백 받은 결과를 도출
