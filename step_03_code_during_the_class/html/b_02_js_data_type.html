웹 기초 - Javascript

데이터 타입(형타입)

  • 숫자: js에서는 실수, 정수 구분없이 숫자
  • 문자: 문자는 반드시 '' 내부에 글자를 작성하는 형태이며, 숫자로 보이더라도 '' 내부에 있으면, 문자
  • 논리형: 참(true), 거짓(false)
  • undefined: 값이 지정되지 않은 상태, 변수나 그 무엇은 존재/유지하는 데이터 형태
  • null: 값이 없거나, 값 전달시 없는 값

특수형

  • 객체(배열/객체)
    • 배열: html 문서에서 말하는 ol 처럼 여러개의 목록으로 대괄호 [ ] 안에 내용을 나열하는 것
    • 객체: 배열처럼 나열하기는 하지만 순서는 없고, 해당하는 목록에 대한 정의를 하는 것으로 html의 dl/css의 속성 형태의 개념과 같다. 중괄호 {} 내부에 여러개의 내용을 담아서 그룹화시킨다.
  • 함수: 바로 실행시키거나 내용을 담는 기능이라기보다는 어떠한 행동을 처리하기 위해 만들어놓은 묶음
    (선언식, 표현식, 즉시실행함수)

다른 프로그래밍 언어와 다른점 중 대표적 특징 하나는 객체 내부에 배열을 담거나, 함수를 담을 수 있다는 것이다.
